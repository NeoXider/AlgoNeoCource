# Пример Quiz — Single Choice

Тест с одним правильным ответом и краткое руководство по использованию блока `quiz`.

## Как использовать

- Вставьте в Markdown fenced-блок `quiz`.
- Укажите `id`, `kind`, `text` и `answers`.
- Для `single` должен быть ровно один вариант с `correct: true`.

## Что важно помнить

- `id` должен быть уникален в рамках курса.
- Ответ отправляется по клику.
- Попытка фиксируется сразу.
- Состояние квизов сохраняется локально автоматически.
- Настройки доступны в `Tools -> AlgoNeoCourse -> Settings -> Open Quiz Settings`.
- Пояснение лучше выносить на следующий слайд. [▶](unity://slide?dir=next)

## Пример

```quiz
id: sc-hello
kind: single
text: Какой модификатор доступа делает поле приватным в C#?
answers:
  - text: private
    correct: true
  - text: public
  - text: protected
  - text: internal
```

Следующая страница может содержать объяснение.

---

## Пояснение (пример)

- Почему `private`: этот модификатор ограничивает доступ к полю внутри класса.
- Почему не `public`: он делает поле доступным извне.
- Почему не `protected`: он открывает доступ наследникам.
- Почему не `internal`: он ограничивает доступ сборкой, а не только классом.
