# Тематический план обучения на Unity 2 год

## Краткое описание курса  
Курс «Разработка игр на Unity: 2 год обучения» – практическая программа, в которой участники поэтапно осваивают язык C#, игровой движок Unity и ключевые аспекты создания игровых проектов. Программа рассчитана на подростков 13–15 лет, уже знакомых с Unity, но без опыта программирования.

## Целевая аудитория  
- Начинающие разработчики 13–15 лет.  
- Пользователи Unity, умеющие создавать проекты, но не владеющие C#.

## Что будут изучать участники  

### Основы программирования на C#  
- Работа с переменными, условиями, циклами, функциями и классами.  
- Принципы объектно‑ориентированного программирования (ООП).  
- Чтение и написание кода для игровых механик.

### Разработка игр в Unity  
- Создание 2D‑ и 3D‑игр с нуля.  
- Работа с физикой, анимацией, звуками и визуальными эффектами.  
- Разработка интерфейсов (меню, здоровье, очки).  
- Системы сохранения прогресса и загрузки уровней.

### Работа в команде и проектный подход  
- Планирование задач и распределение ролей в разработке.  
- Презентация проектов и защита своих решений.

## Навыки, которые приобретут участники  

### Hard Skills  
- Навык программирования на C# (уровень Junior Developer).  
- Работа в Unity: создание сцен, настройка объектов, управление анимацией с помощью кода.  
- Понимание игровых механик (физика, ИИ, генерация уровней).  
- Основы работы с Git и GitHub для контроля версий.

### Soft Skills  
- Логическое мышление – разбор сложных задач на этапы.  
- Креативность – проектирование уникальных игровых решений.  
- Работа в команде (для групповых проектов).  
- Презентационные навыки – защита своих проектов.
