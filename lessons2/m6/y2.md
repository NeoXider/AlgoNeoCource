# Модуль 6, Урок 2: Invoke и спавнеры

**Цель урока:** научиться вызывать методы по таймеру через `InvokeRepeating`, добавить спавнер астероидов и обработать попадание пули.

---

Когда объекты появляются сами по времени, игра живёт без постоянного участия игрока. Для этого нужны таймеры и спавнеры.

---

## Проблема урока: игрок пока стреляет в пустоту

После первого урока корабль уже двигается и стреляет.
Но игры ещё нет: на сцене нет угрозы и нет цели для выстрела.

Сегодня добавим систему, которая сама создаёт астероиды, и научим пулю уничтожать их при попадании.

---

## Теория 1: Вызов метода по времени — InvokeRepeating

Unity позволяет вызывать метод не сразу, а через паузу, и повторять вызов через заданный интервал.

```csharp
InvokeRepeating("ИмяМетода", задержкаДоПервогоВызова, интервалПовторения);
```

Пример: вызывать метод `SpawnAsteroid` первый раз через 1 секунду, потом каждые 1,5 секунды:

```csharp
void Start()
{
    InvokeRepeating(nameof(SpawnAsteroid), 1f, 1.5f);
}

void SpawnAsteroid()
{
    // создаём астероид
}
```

> **Как работает InvokeRepeating по шагам:**
>
> 1. В `Start()` один раз вызывается `InvokeRepeating(nameof(SpawnAsteroid), 1f, 1.5f)`.
> 2. Unity ждёт 1 секунду, затем первый раз вызывает метод `SpawnAsteroid()`.
> 3. После этого Unity вызывает `SpawnAsteroid()` каждые 1.5 секунды, пока объект активен.
> 4. Внутри `SpawnAsteroid` вы создаёте новый астероид — так получается спавнер по таймеру.

`nameof(SpawnAsteroid)` — это имя метода в виде строки, так удобнее и безопаснее, чем писать `"SpawnAsteroid"` вручную.

## Теория 2: Спавнер и случайная позиция

**Спавнер** — объект, который создаёт другие объекты (астероиды, врагов) по таймеру.

Чтобы объекты появлялись в разных местах по горизонтали, используют случайное число:

```csharp
float randomX = Random.Range(minX, maxX);
Vector3 position = new Vector3(randomX, spawnY, 0f);
Instantiate(prefab, position, Quaternion.identity);
```

`Random.Range(minX, maxX)` даёт случайное число в диапазоне от minX до maxX.

Атрибуты Unity помогают сделать инспектор понятнее:

```csharp
[Header("Спавн астероидов")]
[SerializeField] private float asteroidRate = 1.5f;
```

`[Header("...")]` — рисует подзаголовок в инспекторе над полем.

## Практика 2: Спавнер астероидов

### Шаг 2: Создайте объект спавнера

Создайте пустой объект `Spawner`, добавьте скрипт `TimedSpawner.cs`.

### Шаг 3: Добавьте таймер и создание астероида

В `Start()` вызовите `InvokeRepeating(nameof(SpawnAsteroid), startDelay, asteroidRate)`.
В методе `SpawnAsteroid()` вычислите позицию через `Random.Range(minX, maxX)` и `Instantiate(asteroidPrefab, position, Quaternion.identity)`.

**Пример TimedSpawner (только астероиды):**

```csharp
[Header("Астероиды")]
[SerializeField] private GameObject asteroidPrefab;
[SerializeField] private float minX = -7f, maxX = 7f, spawnY = 6f;
[SerializeField] private float asteroidStartDelay = 1f, asteroidRate = 1.5f;

void Start()
{
    InvokeRepeating(nameof(SpawnAsteroid), asteroidStartDelay, asteroidRate);
}

void SpawnAsteroid()
{
    float randomX = Random.Range(minX, maxX);
    Vector3 position = new Vector3(randomX, spawnY, 0f);
    Instantiate(asteroidPrefab, position, Quaternion.identity);
}
```

## Теория 3: Попадание — это разговор двух объектов

Когда пуля касается астероида, Unity вызывает `OnTriggerEnter2D`.
Внутри этого метода один объект может спросить другой:

```csharp
other.GetComponent<ProjectileMover>()
```

Если компонент найден, значит астероид столкнулся именно с пулей игрока.
Тогда можно уничтожить оба объекта:

```csharp
Destroy(other.gameObject);
Destroy(gameObject);
```

## Пояснение

- `GetComponent<ProjectileMover>()` возвращает компонент пули, если он есть на объекте столкновения. Если его нет, результат будет `null`.

## Практика 3: Астероид с уничтожением от пули

**Задача:**

### Шаг 1: Настройте столкновение

1. На префаб астероида добавьте Collider (2D) и включите Is Trigger.

### Шаг 2: Добавьте реакцию на пулю

В скрипте астероида в `OnTriggerEnter2D(Collider2D other)` проверьте, есть ли у `other` компонент пули игрока (`GetComponent<ProjectileMover>()`).
Если да — уничтожьте и пулю, и астероид: `Destroy(other.gameObject); Destroy(gameObject);`

```csharp
void OnTriggerEnter2D(Collider2D other)
{
    if (other.GetComponent<ProjectileMover>() != null)
    {
        Destroy(other.gameObject);
        Destroy(gameObject);
    }
}
```

## Если попадание не срабатывает

Это самая частая проблема урока.
Проверьте не код, а сцену:

- у пули есть `Collider2D`;
- у астероида есть `Collider2D`;
- хотя бы у одного из них включён `Is Trigger`;
- хотя бы у одного объекта есть `Rigidbody2D`;
- скрипт `ProjectileMover` висит именно на пуле.

Если хотя бы одного пункта нет, `OnTriggerEnter2D` может не вызваться.

```csharp

```
