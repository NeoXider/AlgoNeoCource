# M6 Result Scripts

Папка хранит контрольное состояние C#-скриптов на конец каждого урока модуля 6.

- `y1` — управление кораблём и первый выстрел.
- `y2` — спавнер астероидов и попадание пули.
- `y3` — `GameManager`, очки, Game Over и рост сложности.
- `y4` — полный финальный набор скриптов модуля.

У каждого урока свой namespace: `Lessons.M6.Y1`, `Lessons.M6.Y2`, `Lessons.M6.Y3`, `Lessons.M6.Y4`.
Так папки можно переносить в Unity для проверки без конфликта одинаковых имён классов.

Важно: если переносите конкретный урок в Unity, добавляйте компоненты из namespace этого урока.
Например, для первого урока используйте `Lessons.M6.Y1.PlayerShipController`.
