# M4 Result Scripts

Папка хранит контрольное состояние C#-скриптов на конец каждого урока модуля 4.

- `y1` — поле, клетки, массив и первый клик.
- `y2` — очередь ходов, X/O и блокировка занятых клеток.
- `y3` — `List`, очистка поля, победа и ничья.
- `y4` — бот и временный режим игры против бота.
- `y5` — полный финальный набор скриптов модуля: выбор режима, статус, рестарт поля и финальная UI-логика.

У каждого урока свой namespace: `Lessons.M4.Y1`, `Lessons.M4.Y2` и так далее.
Так папки можно переносить в Unity для проверки без конфликта одинаковых имён классов.

В `y5` показ стартового экрана перенесён в `Start()`: `Awake()` только готовит ссылки и подписки, а стартовая логика выполняется после подготовки объектов сцены.
