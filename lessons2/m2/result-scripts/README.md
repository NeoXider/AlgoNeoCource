# M2 Result Scripts

Папка хранит контрольное состояние C#-скриптов на конец каждого урока модуля 2.

- `y1` — первый скрипт и вывод в Console.
- `y2` — переменные кликера и публичные поля в Inspector.
- `y3` — кликабельный объект, штраф, размер и UI-счёт.
- `y4` — методы, улучшение силы клика и смена спрайта через кнопку.
- `y5` — полный финальный набор скриптов модуля: кликер со звуком, эффектом и сменой спрайта.

У каждого урока свой namespace: `Lessons.M2.Y1`, `Lessons.M2.Y2` и так далее.
Так папки можно переносить в Unity для проверки без конфликта одинаковых имён классов.
