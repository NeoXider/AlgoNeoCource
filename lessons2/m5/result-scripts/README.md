# M5 Result Scripts

Папка хранит контрольное состояние C#-скриптов на конец каждого урока модуля 5.

- `y1` — результат после урока 1.
- `y2` — результат после урока 2.
- `y3` — результат после урока 3.
- `y4` — результат после урока 4.
- `y5` — результат после урока 5.
- `y6` — полный финальный набор скриптов модуля.

У каждого урока свой namespace: `Lessons.M5.Y1`, `Lessons.M5.Y2` и так далее.
Так папки можно переносить в Unity для проверки без конфликта одинаковых имён классов.

Важно: если переносите конкретный урок в Unity, добавляйте компоненты из namespace этого урока.
Например, для первого урока используйте `Lessons.M5.Y1.CrossbowShooter`.
