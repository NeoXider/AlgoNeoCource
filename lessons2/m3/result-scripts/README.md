# M3 Result Scripts

Папка хранит контрольное состояние C#-скриптов на конец каждого урока модуля 3.

- `y1` — условия, диагностика и первое движение Dino.
- `y2` — `else if`, составные условия, прыжок только с земли и камера.
- `y3` — циклы, препятствия, prefab и спавнер.
- `y4` — полный финальный набор скриптов модуля: Game Over, UI-счёт, Restart и остановка игры.

У каждого урока свой namespace: `Lessons.M3.Y1`, `Lessons.M3.Y2`, `Lessons.M3.Y3`, `Lessons.M3.Y4`.
Так папки можно переносить в Unity для проверки без конфликта одинаковых имён классов.
